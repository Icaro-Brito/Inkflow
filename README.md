# 🖋️ Inkflow

[![Deploy Database to Aiven](https://github.com/CodeByWesley/Inkflow/actions/workflows/deploy-db.yml/badge.svg)](https://github.com/CodeByWesley/Inkflow/actions/workflows/deploy-db.yml)

🚧 Projeto em desenvolvimento

Sistema full stack de agendamento para estúdios de tatuagem, desenvolvido com foco em resolver problemas reais de gestão de sessões, serviços e orçamentos.

---

## 📌 Sobre o projeto

O **Inkflow** é uma aplicação que simula o funcionamento de um estúdio de tatuagem, permitindo o gerenciamento completo de agendamentos, serviços e valores.

O projeto foi criado com o objetivo de consolidar conhecimentos em desenvolvimento backend com Java, integração com banco de dados em nuvem e construção de aplicações full stack próximas de cenários reais do mercado.

---

## 🚀 Tecnologias utilizadas

- **Java (Spring Boot)**
- **HTML, CSS e JavaScript**
- **SQL**
- **Aiven (Cloud Database)**
- **GitHub Actions (CI/CD)**

---

## ⚙️ Funcionalidades

- 📅 Gerenciamento de agendamentos  
- 💰 Cálculo de orçamento de serviços  
- 🧾 Cadastro e listagem de serviços  
- 🔗 Integração entre frontend e backend  
- ☁️ Persistência de dados em banco na nuvem  

---

## 🔄 Status do projeto

O projeto está em desenvolvimento ativo, com melhorias contínuas na arquitetura, regras de negócio e experiência do usuário.

## 🛠️ Como executar o projeto

```bash
# Clone o repositório
git clone https://github.com/CodeByWesley/Inkflow.git

# Acesse a pasta
cd Inkflow

# Execute a aplicação
./mvnw spring-boot:run
```

> ⚠️ Configure as variáveis de ambiente para conexão com o banco de dados (Aiven) antes de executar.

---

## ☁️ Deploy e Banco de Dados

A aplicação utiliza banco de dados em nuvem via **Aiven**, com pipeline automatizado utilizando **GitHub Actions** para integração e deploy.

Essa abordagem permite simular um ambiente profissional, incluindo práticas de CI/CD e gerenciamento de infraestrutura.
